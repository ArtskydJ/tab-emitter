tab-emitter

tab-emitter is a client-side javascript module that allows
you to send events between browser tabs/windows. The
sending/recieving web pages must have the same origin.

tab-emitter is written to work with browserify, and is
extremely easy to implement. Give it a shot, or if you're not
convinced, play with this demo!

Open this site in multiple tabs/windows. Try placing the
windows beside each other.
